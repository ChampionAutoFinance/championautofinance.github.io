# ContactCard Deploy

Expected URL:
- `/ContactCard/`

Files inside this folder:
- `index.html`
- `style.css`
- `script.js`
- `Champion-Auto-Finance.vcf`
- `assets/background.jpg`
- `assets/logo.png`
- `assets/favicon-tab.jpg`
