# Champion Auto Finance GitHub Pages

Sanitized static Pages build from the appdata backup.

Included:
- Root portal page
- `worksheet/`
- `structure/`
- `dealertools/`
- `ContactCard/`
- `ContactCard-Jason/`
- `ContactCard-Joe/`

Excluded:
- WordPress PHP runtime
- `wp-config.php`
- SSL keys and certificates
- Logs, cPanel data, and private server files

GitHub Pages serves static files only, so the original WordPress/PHP site was not published from this backup.
