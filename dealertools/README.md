# Champion Actions Page

Static landing page for Champion Auto Finance dealer actions. The visual style is matched to the live Champion Auto Finance homepage.

## Files

- `index.html` - page markup
- `styles.css` - homepage-matched layout, hero background, nav, buttons, trust row, and footer
- `script.js` - dummy Credit Application notice
- `assets/champion-background.png` - homepage hero background from `championautofinance.com`
- `assets/champion-site-logo.webp` - homepage navigation logo from `championautofinance.com`
- `assets/champion-shield.png` - homepage shield image from `championautofinance.com`

## Links

- Deal Structure opens `../structure/`
- Credit Application is a placeholder button for now

This project has no build step. Upload the folder contents wherever the page should live, or publish the folder as a static GitHub Pages site.
